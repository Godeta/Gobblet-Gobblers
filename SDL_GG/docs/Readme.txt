Projet réalisé par Arnaud GODET étudiant à l'IUT GRAND OUEST NORMANDIE en Janvier 2020

Dossiers : 
- bin contient le necessaire pour executer le programme (tous les .dll) et l'executable
- data contient les images et polices utilisées
- docs la documentation rédigée par mes soins
- include contient tous les .h necessaires
- lib contient les librairies nécessaires à la SDL 1.2
- obj contient les .o pour linux
- sources contients le code en lui même

Le reste est le nécessaire pour code bloc (le fichier de projet, les dépendances et l'affichage)
